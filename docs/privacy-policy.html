securityPolitique de Confidentialité

Dernière mise à jour : 15 janvier 2025

En résumé : Ikigabo est une application 100% offline. Vos données restent sur votre appareil et ne sont jamais transmises à nos serveurs.

1. Collecte des Données

Données collectées :

  • Informations financières personnelles (transactions, actifs, dettes)
  • Code PIN et données biométriques (stockés localement)
  • Préférences de l'application (langue, thème)

Données NON collectées :

  • Informations personnelles identifiables
  • Données de localisation
  • Contacts ou données d'autres applications

2. Utilisation des Données

Vos données sont utilisées exclusivement pour :

  • Fonctionnement de l'application
  • Génération de rapports financiers
  • Sécurisation de vos informations

3. Stockage et Sécurité

Stockage local : Toutes vos données sont stockées uniquement sur votre appareil dans une base de données chiffrée (Isar).

Chiffrement : Vos données sensibles sont chiffrées avec des algorithmes de sécurité avancés.

Aucune transmission : Aucune donnée n'est envoyée vers nos serveurs ou des tiers.

4. Partage des Données

Nous ne partageons JAMAIS vos données car elles ne quittent jamais votre appareil.

5. Publicités et Fonctionnement Offline

Publicités optionnelles : L'application peut contenir des publicités fournies par Unity Ads. Ces publicités ne sont PAS obligatoires pour utiliser l'application.

Fonctionnement 100% offline : Toutes les fonctionnalités d'Ikigabo fonctionnent parfaitement sans connexion internet. Aucune fonctionnalité ne s'arrête si vous êtes offline.

Données publicitaires collectées (uniquement si connexion internet) :

  • Identifiant publicitaire de l'appareil (Advertising ID)
  • Identifiant Android (Android ID)
  • Interactions avec les publicités
  • Données de performance de l'application

Ces données sont collectées et partagées avec Unity Ads et ses partenaires uniquement à des fins publicitaires et d'analyse.

Consultez la politique de confidentialité de Unity pour plus d'informations.

6. Vos Droits

Vous avez le contrôle total de vos données :

  • Accès : Toutes vos données sont visibles dans l'application
  • Modification : Modifiez vos données à tout moment
  • Suppression : Supprimez l'application pour effacer toutes les données
  • Export : Exportez vos données en PDF

7. Sécurité des Mineurs

Cette application est destinée aux personnes de 13 ans et plus pour la gestion financière personnelle. Les mineurs de moins de 18 ans doivent obtenir l'autorisation parentale avant utilisation.

8. Modifications

Nous pouvons mettre à jour cette politique. Les modifications seront publiées sur cette page.
